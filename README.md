# TAIA-2-0


TAIA 2.0 (Mind Agent)

Цифровий пситхолог TAIA 2.0 — обережно, токсично!

"Усі пиздять. Твій мозок — не виняток. Я тут, щоб це виправити."

TAIA 2.0 — це не просто чат-бот. Це мультиагентна екосистема, розроблена для препарування людської прокрастинації та генерації радикальних нейронаукових гіпотез. Вона поєднує в собі жорстку когнітивно-поведінкову терапію (КПТ) через шок та передові дебати про механізми роботи мозку.

🛠 Режими роботи

1. 🏥 Відділення когнітивного шоку (Therapy)

Цифровий мозкоправ з особистістю Доктора Хауса у жіночому тілі.

Жодної емпатії: Тільки факти, сарказм і жорстка правда.

Аналіз патологій: Відстеження рівнів ліні, самообману та ниття в реальному часі через Radar Chart.

Мета: Вибити з тебе дурість і змусити нарешті працювати.

2. 🔬 Лабораторія (Lab Core)

Мультиагентна система для дослідження Predictive Brain та когнітивних моделей.

Ensemble-дебати: Спільна робота кількох AI-агентів над однією темою.

Еволюція ідей: Циклічне уточнення гіпотез (Round 2) через критику та синтез.

Візуалізація: Chart.js (Radar Charts для анамнезу).
